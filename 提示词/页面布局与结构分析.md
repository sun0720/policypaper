## 一、整体架构

```
┌──────────────────────────────────────────────┐
│  Header (sticky, 始终固定在顶部)              │
│  Logo + 品牌名 "PolicyPaper"                  │
├──────────────────────────────────────────────┤
│                                              │
│  Main Content (max-width: 1200px, 居中)      │

**关键文件**：`src/app/layout.tsx`

- `<html lang="zh-CN">`
- Header 使用 `position: sticky; top: 0; z-index: 100`，带 `backdrop-filter: blur(8px)` 毛玻璃效果

- Footer 顶部有渐变分隔线（`linear-gradient(90deg, transparent 0%, var(--divider) 20%, var(--divider) 80%, transparent 100%)`）

**关键文件**：`src/app/page.tsx`

1. `getAllDates()` → 获取所有日期列表（用于侧边栏）

3. 遍历日期 → 渲染 `NewsCard`

## 五、6 个核心组件详解

### 5.1 NewsCard（新闻摘要卡片）

**文件**：`src/components/NewsCard.tsx`

```
┌─────────────────────────────────────────┐
│ [产业经济]                              │  ← FieldBadgeSm
│ 新闻标题                        → 原文  │  ← h2 + hover 显示的原文链接
│ 正文摘要前150字...                      │  ← .news-excerpt (3行截断)
│ ▶ 🎓 5个论文选题                       │  ← <details> 可折叠
│   1. 宏观视角：xxx                      │
│   2. 微观视角：xxx                      │

**样式特点**：
- 白色卡片 + 圆角 10px + 阴影

- `<details>` 原生折叠组件，无需 JS

## 六、响应式断点

### 断点 1：≤900px（平板竖屏）

- 双栏 → 单栏（`flex-direction: column`）
- 侧边栏日期列表从纵向变为横向 wrapping 标签
- Hero 区域文字居中
- 侧边栏 `position: static`，宽度 100%

### 断点 2：≤768px（平板/大手机）

- `.prose` 内边距缩小：`2rem → 0.75rem`
- 正文字号：`17px → 16px`
- TopicCard 内边距收紧，左边框从 3px → 2px
- NewsCard 内边距收紧：`1.25rem 1.5rem → 1rem 1.1rem`
- TopicCard info table 左列宽度：`100px → 72px`

### 断点 3：≤480px（手机）

- 正文字号：`16px → 15px`
- 标题字号缩小
- **TopicCard info table**：从双列 table → block 堆叠（左列标签在上，右列内容在下），左列变 accent 色
- NewsCard 圆角缩小：`10px → 8px`
- 日期标题行：水平排列 → 纵向堆叠
- 底部导航：水平三栏 → 纵向三行居中
- 新闻正文段首缩进：`2em → 1.5em`

## 七、数据层架构

**关键文件**：`src/lib/data.ts`、`src/lib/parser.ts`

```
data/exports/*.md  ──parse──→  DailyExport[]  ──缓存 60s──→  查询函数
```

- **数据源**：`data/exports/YYYY-MM-DD-paper-topics.md` Markdown 文件
- **解析器**：`parseDailyExport()` 解析 Markdown 为结构化数据
- **缓存**：内存缓存 60 秒（`CACHE_TTL = 60_000`），避免每次请求重新读取文件系统
- **查询函数**：
  - `getAllDates()` — 首页侧边栏
  - `getByDate(date)` — 日期页
  - `getByField(field)` — 分类页
  - `getNewsBySlug(slug)` — 详情页
  - `getArchiveData()` — 归档页
  - `getAllFields()` — FilterBar

**DailyExport 数据结构**：
```typescript
interface DailyExport {
  date: string;           // "2026-06-04"
  newsCount: number;
  topicsCount: number;
  fields: string[];       // 覆盖的经济领域
  news: NewsData[];       // 新闻列表
}

interface NewsData {
  slug: string;           // URL slug
  title: string;          // 新闻标题
  url: string;            // 原文链接
  content: string;        // 正文
  economicField: string;  // 经济领域
  subField?: string;      // 子领域
  topics: TopicData[];    // 论文选题列表
}

interface TopicData {
  sortOrder: number;           // 1-5
  perspective: string;         // "宏观视角"
  title: string;               // 论文标题
  researchQuestion: string;    // 研究问题
  theoreticalFramework?: string;
  methodology?: string;
  dataSources?: string;
  innovation?: string;
  researchApproach?: string;   // 🔬 研究思路（Markdown）
}
```

## 八、设计亮点总结
