# 网页布局优化 Prompt

## 优化提示

优化网站的页面布局和视觉设计，提升学术阅读体验。当前网站已具备基础排版（暖白底、衬线标题、16px/1.8 行高正文），需要从以下方向改进：

### 优化方向

1. **信息密度**：首页 DateCard 列表间距是否合适？选题预览是否需要更紧凑的呈现？

2. **视觉层级**：标题、正文、元数据的字号和颜色层级是否足够区分？TopicCard 内部的 info table 是否可读性良好？

3. **导航体验**：是否需要面包屑之外的导航方式（如侧边栏、目录跳转）？

4. **暗色模式**：当前暗色模式配色是否舒适？需要调整哪些颜色？

5. **响应式**：移动端布局是否需要 special 处理？选题卡片在小屏上是堆叠还是保持表格布局？

### 约束

- **框架**：Next.js 16 App Router
- **样式方案**：Tailwind CSS 4 + CSS 变量（当前系统字体栈，零网络字体加载）
- **排版不动**：保留衬线标题 + 无衬线正文的系统字体栈
- **配色方向**：保留暖白纸张底色 + 琥珀棕强调的学术风格

### 交付物

- 逐个页面的布局调整方案（文字描述或代码实现）
- 调整后的 globals.css 和组件代码

### 优先级

你认为哪方面最需要改善？可以先从 1-2 个方向开始。
