#### 1. 📊 新古典视角 — 碳排放权交易对企业全要素生产率的影响——基于A股上市公司的准自然实验

| | |
|---|---|
| **研究问题** | 中国碳排放权交易市场的建立是否通过价格信号引导企业提升了绿色全要素生产率？碳价波动对企业减排投资决策的边际影响有多大？ |
| **理论框架** | 庇古税与科斯定理（排污权交易）；波特假说（Porter & van der Linde, 1995）；异质性企业模型（Melitz, 2003） |
| **研究方法** | 渐进DID（利用碳市场分批次纳入行业的准自然实验）；OP/LP法测算企业绿色TFP；事件研究法检验碳价波动的市场反应 |
| **数据来源** | CSMAR（A股上市公司财务与能耗数据）；上海环境能源交易所（碳配额价格与成交量）；国家统计局（行业碳排放数据） |
| **创新点** | 首次利用碳市场分批扩容的渐进DID设计识别碳定价对企业TFP的因果效应，区分配额交易与CCER的不同影响渠道 |

#### 2. 🏛️ 制度经济学视角 — 生态文明制度体系的激励相容——生态补偿、环保督察与地方政府行为

| | |
|---|---|
| **研究问题** | 生态补偿转移支付和中央环保督察两类制度工具，哪个对地方政府环境治理行为产生了更强的激励效应？两者的交互作用是互补还是替代？ |
| **理论框架** | 激励相容机制设计（Hurwicz, 1973）；财政联邦主义（Qian & Weingast, 1997）；多任务委托代理模型（Holmstrom & Milgrom, 1991） |
| **研究方法** | 省级面板固定效应 + 交互项（生态补偿×环保督察强度）；断点回归（利用环保督察进驻的准随机时间）；文本分析法量化政府环境关注度 |
| **数据来源** | 财政部（生态补偿转移支付）；生态环境部（环保督察通报案例）；各省政府工作报告（文本分析）；国家统计局（省级环境质量面板） |
| **创新点** | 首次在同一框架中比较生态补偿（正向激励）与环保督察（负向约束）的制度有效性，检验两者的交互效应 |

#### 3. 🧠 行为经济学视角 — 绿色消费的「态度-行为」缺口——信息干预与社会规范的随机对照实验

| | |
|---|---|
| **研究问题** | 中国消费者绿色消费「态度-行为」缺口的规模和成因是什么？碳标签信息干预和社会规范助推哪个更有效缩小这一缺口？ |
| **理论框架** | 态度-行为缺口理论（Carrington et al., 2010）；助推理论（Thaler & Sunstein, 2008）；社会规范理论（Cialdini et al., 1990） |
| **研究方法** | 在线随机对照实验（2×2因子设计：碳标签×社会规范）；中介效应分析检验心理机制；异质性分析按人口特征分组 |
| **数据来源** | 自主设计在线RCT平台（Credamo/见数）；CFPS（家庭消费与环境态度模块）；淘宝/京东绿色商品销售数据 |
| **创新点** | 首次利用大规模RCT在中国消费者群体中比较碳标签与社会规范助推的效果差异，量化态度-行为缺口的规模并识别心理中介机制 |

#### 4. ⚖️ 政治经济学视角 — 环境规制的区域差异与污染避难所效应——来自中国制造业企业选址的证据

| | |
|---|---|
| **研究问题** | 各省环境规制强度差异是否导致了污染产业从高规制地区向低规制地区转移？这种转移多大程度由「逐底竞争」驱动？ |
| **理论框架** | 污染避难所假说（Copeland & Taylor, 1994）；逐底竞争理论（Oates, 1972）；环境联邦主义 |
| **研究方法** | 条件Logit模型估计企业选址决策（区位选择作为环境规制强度的函数）；空间杜宾模型检验污染产业空间集聚；DID检验环保督察前后选址变化 |
| **数据来源** | 国家统计局（工业企业数据库 + 污染排放）；生态环境部（各省环境规制强度指数）；天眼查（企业注册地址与行业分类） |
| **创新点** | 首次利用微观企业选址数据识别污染避难所效应，区分「逐底竞争」与「比较优势」两种机制，评估中央督察的遏制效果 |

#### 5. 🌏 发展经济学视角 — 绿色转型与经济增长的协同路径——环境库兹涅茨曲线的再检验

| | |
|---|---|
| **研究问题** | 中国城市层面经济增长与环境质量是否仍呈倒U型？绿色技术创新和产业升级是否使「拐点」提前到来？ |
| **理论框架** | 环境库兹涅茨曲线（Grossman & Krueger, 1995）；结构转型理论（Kuznets事实）；诱导性技术变迁（Acemoglu et al., 2012） |
| **研究方法** | 地级市面板固定效应 + 二次项；门槛回归识别结构转型的拐点效应；中介效应检验绿色专利的中介作用 |
| **数据来源** | 国家统计局（地级市GDP、产业结构）；生态环境部（PM2.5、SO2、CO2）；国家知识产权局（绿色专利）；DMSP/VIIRS夜间灯光 |
| **创新点** | 首次在统一框架中检验绿色技术创新和产业结构升级对EKC拐点位置的因果影响，为中国城市绿色转型提供实证路径 |
