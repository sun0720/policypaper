## 李强签署国务院令 公布《退役军人就业创业促进条例》

> 📂 **经济领域**：劳动经济 — 就业政策
> 🔗 **原文链接**：https://www.gov.cn/yaowen/liebiao/202606/content_7073794.htm

**📄 新闻正文**

新华社北京6月30日电 国务院总理李强日前签署国务院令，公布《退役军人就业创业促进条例》（以下简称《条例》），自2026年8月1日起施行。《条例》旨在促进退役军人就业创业，发挥退役军人人力资源作用，维护退役军人合法权益，让军人成为全社会尊崇的职业。《条例》共6章48条，主要规定了以下内容。一是明确总体要求。退役军人就业创业工作坚持普惠与优待叠加的原则，采取政府推动、市场引导、社会支持相结合的方式，鼓励和扶持退役军人就业创业。引导退役军人树立正确的择业观念，通过教育培训、就业支持、创业扶持等措施，促进退役军人高质量充分就业。明确有关单位的职责分工。二是确定教育培训政策。国家采取措施加强对退役军人的教育培训。退役军人参加国家教育考试和接受学历教育时按照国家有关规定享受优待和资助政策。国家将退役军人职业技能培训纳入终身职业技能培训政策和组织实施体系，对适应性培训、免费职业技能培训和培训补贴等提出要求。三是完善就业优待措施。机关、群团组织、事业单位和国有企业在招录或者招聘人员时，对退役军人的年龄和学历条件可以适当放宽，同等条件下优先招录、招聘退役军人。各地设置一定数量的基层公务员职位，面向服现役满5年的高校毕业生退役军人招考。明确艰苦边远地区县乡事业单位招聘、国家综合性消防救援队伍消防员等特殊岗位招录时对退役军人的优待政策。四是优化创业扶持举措。明确对退役军人创业给予税收、融资、信贷等扶持政策。县级以上地方人民政府支持引导有创业意愿和培训需求的退役军人参加创业培训。鼓励退役军人创业创新。此外，对违反《条例》规定的有关行为明确了相应法律责任。

### 🎓 论文选题

#### 1. 👷 劳动经济学(工资/就业/人力资本) — 退役军人就业优待政策的人力资本回报率：基于"军转民"技能可迁移性的准实验研究

| | |
|---|---|
| **研究问题** | 《条例》规定的学历放宽、优先招录等就业优待措施能否有效提高退役军人的就业质量（工资水平、岗位匹配度）？军事技能向民用岗位的可迁移性如何影响人力资本回报率？ |
| **理论框架** | Mincer人力资本工资方程（Mincer, 1974）、技能可迁移性理论（Cohen & Haberfeld, 1997）、信号传递模型（Spence, 1973） |
| **研究方法** | 模糊断点回归（Fuzzy RDD）+ 倾向得分匹配双重差分（PSM-DID） |
| **数据来源** | 退役军人事务部退役军人就业登记数据库、各省事业单位公开招聘公告与录用名单（2020-2028年）、中国劳动收入动态调查（CLDS）、中国综合社会调查（CGSS） |
| **创新点** | 首次测度中国退役军人军事技能的民用迁移率，将"军龄×技能类型"交互项纳入Mincer方程，识别技能折旧对人力资本回报的动态影响 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于Mincer（1974）人力资本理论和Spence（1973）信号模型，退役军人面临双重挑战：①军事专用性人力资本（如武器操作）在民用市场存在"技能折旧"（H1：军龄越长、技能专用性越高，初始就业质量越低）；②"退役军人"标签具有信号价值——纪律性、执行力等可迁移技能的正向信号（H2：《条例》实施后，退役军人工资溢价上升）；③优待政策的"年龄放宽"条款降低了退役军人的求职搜寻成本（H3：就业等待期缩短但岗位匹配度可能下降）。

**2. 识别策略**
利用《条例》2026年8月实施的时间断点，以2025年8月至2026年7月退役的军人为对照组、2026年8月至2027年7月退役的军人为处理组，构建DID模型。为缓解选择性偏误，进一步以退役安置地的"国有经济就业占比"作为工具变量——该变量影响优待政策执行力度但不直接影响个体能力，满足排他性约束。

**3. 计量模型设定**
基准模型：$\ln W_{it} = \alpha + \beta_1 Treat_i \times Post_t + \beta_2 Edu_i + \beta_3 MilYears_i + \beta_4 SkillTransfer_i + \gamma X_{it} + \delta_r + \lambda_t + \epsilon_{it}$，其中$\ln W_{it}$为退役军人$i$在$t$期的对数工资，$Treat_i \times Post_t$为政策处理效应，$MilYears_i$为服役年限，$SkillTransfer_i$为技能可迁移性指数，$X_{it}$为控制变量，$\delta_r$为安置地固定效应。

**4. 变量构造与数据**
被解释变量：①月工资收入（CLDS/就业登记数据）；②岗位匹配度指数（职业类型与军事技能相关性评分）；③就业等待天数。核心解释变量：政策虚拟变量（Treat×Post）。技能可迁移性指数：依据O*NET职业数据库与中国军队岗位分类，计算军事岗位与目标民用岗位的"任务重叠度"。控制变量：年龄、学历、军衔、安置地经济发展水平。

**5. 稳健性检验方案**
①更换被解释变量：用岗位等级（管理岗/技术岗/一般岗）替代工资水平；②排除同期政策干扰：控制《职业教育法》修订对退役军人培训补贴的叠加效应；③异质性分析：按兵种（技术兵vs.步兵）、安置地区（东部vs.中西部）分组估计；④动态效应检验：用事件研究法（Event Study）验证政策效应的时间路径，排除"预期效应"；⑤敏感性分析：使用Rosenbaum边界法评估未观测混杂因素的敏感程度。

#### 2. 🏛️ 制度经济学(产权/交易成本/激励) — 普惠与优待叠加原则下的制度激励扭曲：退役军人创业扶持政策的交易成本分析

| | |
|---|---|
| **研究问题** | 《条例》"普惠与优待叠加"的制度设计是否会产生激励扭曲——退役军人身份是否会成为一种"制度租金"，诱发策略性身份寻租？交易成本如何影响政策红利的实际到达率？ |
| **理论框架** | 交易成本理论（Williamson, 1985）、寻租理论（Tullock, 1967; Krueger, 1974）、制度激励相容理论（Hurwicz, 1972机制设计理论） |
| **研究方法** | 制度分析 + 双重差分法（DID）+ 调查实验法（List Experiment） |
| **数据来源** | 各省退役军人事务厅创业扶持审批数据（2022-2028年）、国家市场监管总局企业注册数据库、退役军人创业税收减免申报数据（国家税务总局）、退役军人事务部年度统计公报 |
| **创新点** | 构建"制度租金-交易成本"二维分析框架，量化身份优待政策的"制度套利"规模，揭示政策红利从中央到个体的"漏斗损耗"机制 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于Williamson（1985）交易成本理论与Krueger（1974）寻租理论，"普惠+优待"的制度安排内含张力：①"退役军人"身份附带税收减免、信贷优惠等制度租金，可能诱发非退役军人群体的策略性"身份挂靠"（H1：政策实施后，涉退役军人身份的虚假注册增加）；②申请创业扶持的行政审批环节构成交易成本，导致政策红利的"最后一公里"损耗（H2：交易成本每增加10%，政策到达率下降5-8个百分点）；③地方政府执行《条例》的激励强度与本地财政压力负相关（H3：财政困难地区政策执行率更低）。

**2. 识别策略**
利用各省《条例》实施细则出台的时间差异（中央条例2026年8月生效，各省配套细则预计在2026年Q3-Q4陆续出台），构建交错DID（Staggered DID）模型。以省级退役军人事务厅配套政策发布时间为处理时点，比较先出台与后出台省份的创业注册与税收减免差异。关键假定：各省配套政策出台时间差异与省内创业环境趋势不相关（可通过历史退役军人安置数量作为外生变量验证）。

**3. 计量模型设定**
交错DID模型：$Y_{pt} = \alpha + \sum_{k}\beta_k D_{pt}^k + \gamma X_{pt} + \mu_p + \lambda_t + \epsilon_{pt}$，其中$Y_{pt}$为省份$p$在$t$期的退役军人创业注册数/税收减免总额/政策到达率，$D_{pt}^k$为交错处理虚拟变量（配套政策实施后第$k$期），$X_{pt}$为控制变量（GDP增速、财政收支缺口、营商环境指数），$\mu_p$为省份固定效应，$\lambda_t$为时间固定效应。采用Sun & Abraham（2021）的交互加权估计量纠正异质性处理效应偏误。

**4. 变量构造与数据**
被解释变量：①退役军人新创企业数量（市场监管总局企业注册数据库中标注退役军人法人的企业）；②创业税收减免实际金额（国家税务总局减免税申报数据）；③政策到达率=实际享受优惠人数/符合资格人数。交易成本代理变量：审批环节数、审批平均天数、申请材料份数。控制变量：省级GDP增速、一般公共预算收支缺口、营商环境评价指数、退役军人总量。

**5. 稳健性检验方案**
①安慰剂检验：将处理时点前移2年，检验是否存在"伪处理效应"；②替换估计量：分别使用Callaway & Sant'Anna（2021）和de Chaisemartin & D'Haultfœuille（2020）的稳健DID估计量交叉验证；③排除竞争性政策：控制同期"大众创业万众创新"等普惠性创业扶持政策的干扰；④工具变量法：以各省1955年授衔将军数量作为退役军人制度文化的工具变量；⑤定性补充：对3个典型省份进行半结构化访谈，验证交易成本的微观传导机制。

#### 3. 💰 凯恩斯(总需求/政策乘数) — 退役军人就业创业扶持的财政乘数效应：税收减免与培训补贴的需求拉动机制

| | |
|---|---|
| **研究问题** | 《条例》中税收减免、培训补贴、信贷扶持等财政工具对退役军人就业和消费的刺激效应有多大？不同政策工具（减税vs.补贴vs.信贷）的乘数差异如何？ |
| **理论框架** | Keynes有效需求理论（Keynes, 1936）、财政乘数的状态依赖性（Auerbach & Gorodnichenko, 2012）、定向转移支付的边际消费倾向理论（Slemrod, 1995） |
| **研究方法** | 省级面板向量自回归（Panel VAR）+ 叙事法识别财政冲击 |
| **数据来源** | 各省退役军人事务支出决算数据（财政部）、各省退役军人就业创业补贴发放统计、各省社会消费品零售总额、省级宏观经济季度数据（CEIC）、国家统计局劳动力调查 |
| **创新点** | 首次估算退役军人就业扶持政策的财政乘数，区分"税收减免乘数"、"培训补贴乘数"和"信贷扶持乘数"的异质性效应 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于Keynes有效需求理论与Auerbach & Gorodnichenko（2012）的乘数状态依赖性假说，退役军人就业扶持具有三类需求拉动渠道：①税收减免直接增加退役军人可支配收入，通过边际消费倾向（MPC）拉动消费（H1：退役军人税收减免的MPC约为0.4-0.6，高于一般居民）；②培训补贴提升就业能力，产生长期收入增长预期，促进当期消费（H2：培训补贴乘数>税收减免乘数）；③在经济下行期，退役军人就业补贴的乘数效应因闲置劳动力而放大（H3：失业率每升高1个百分点，乘数增加0.3-0.5）。

**2. 识别策略**
采用叙事法（Romer & Romer, 2010）从国务院及各省退役军人事务厅文件中识别外生性退役军人支出变动——剔除与经济形势相关的"逆周期"支出增加，仅保留制度性政策调整（如补贴标准上调、覆盖面扩大等）带来的支出变动。构建面板VAR模型，利用Cholesky分解识别退役军人财政支出的外生冲击对产出和消费的动态效应。

**3. 计量模型设定**
面板VAR模型：$Y_{pt} = \sum_{j=1}^{J} A_j Y_{p,t-j} + \mu_p + \lambda_t + \epsilon_{pt}$，其中$Y_{pt} = [\Delta VetSpending_{pt}, \Delta Consumption_{pt}, \Delta Employment_{pt}, \Delta GDP_{pt}]'$，$VetSpending$为退役军人就业创业支出（分为税收减免、培训补贴、信贷贴息三个分项）。通过Cholesky分解设定递归识别次序：退役军人支出冲击→消费→就业→GDP。脉冲响应函数追踪8个季度的动态效应，累积乘数为脉冲响应函数的积分。

**4. 变量构造与数据**
退役军人财政支出：各省退役军人事务厅年度决算中"就业创业"科目支出（财政部决算公开数据），细分为税收减免额、培训补贴额、创业贷款贴息额。消费：省级社会消费品零售总额（国家统计局）。就业：退役军人就业率（退役军人事务部统计）。GDP：省级季度实际GDP（CEIC）。叙事法冲击序列：手工收集2018-2026年各省退役军人就业创业政策调整文件，按"非经济周期驱动"标准编码外生支出变动。

**5. 稳健性检验方案**
①替换识别方法：用Blanchard & Perotti（2002）的结构VAR方法替代Cholesky分解；②乘数状态依赖检验：用Auerbach & Gorodnichenko（2012）的平滑转换模型，检验经济扩张期与收缩期的乘数差异；③控制转移支付：在VAR中加入一般性转移支付变量，分离退役军人专项支出的"纯效应"；④交叉验证：用微观调查数据（CFPS/CHFS）估算退役军人家庭的边际消费倾向，与宏观乘数进行一致性检验；⑤样本稳健性：剔除直辖市和自治区（制度环境特殊），仅以普通省份为样本重新估计。

#### 4. 🧠 行为经济学(有限理性/锚定/前景理论) — "让军人成为全社会尊崇的职业"：身份框架效应如何影响退役军人创业决策

| | |
|---|---|
| **研究问题** | 《条例》中"让军人成为全社会尊崇的职业"这一身份框架如何通过社会规范和心理账户影响退役军人的创业风险感知与决策？"尊崇"叙事是否会过度激励高风险创业？ |
| **理论框架** | 前景理论（Kahneman & Tversky, 1979）、框架效应（Tversky & Kahneman, 1981）、社会身份理论（Akerlof & Kranton, 2000）、过度自信理论（Malmendier & Tate, 2005） |
| **研究方法** | 随机对照实验（RCT）+ 实验室博弈实验 + 问卷调查 |
| **数据来源** | 自主设计的退役军人创业决策实验（与3-5个省级退役军人事务厅合作招募实验参与者）、中国家庭追踪调查（CFPS）退役军人子样本、退役军人创业企业存活率数据（市场监管总局） |
| **创新点** | 将Akerlof & Kranton（2000）身份效用模型嵌入创业决策框架，检验"尊崇身份"框架对风险偏好的因果效应，揭示政策叙事与创业失败率之间的隐性关联 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于Kahneman & Tversky（1979）前景理论和Akerlof & Kranton（2000）身份效用理论，"尊崇"叙事对退役军人创业决策产生三重心理效应：①身份框架效应——"军人=强者"的身份认同提升风险容忍度，使参考点右移（H1：身份框架组比控制组多接受15-25%的高风险创业项目）；②社会规范压力——"尊崇"期待可能产生"不创业就是弱者"的隐性压力，导致非理性创业（H2：身份框架组中低能力者的创业比例异常升高）；③过度自信偏差——军事训练中的成功经验被过度泛化到商业领域（H3：有作战经历的退役军人创业失败率高于后勤岗位退役军人）。

**2. 识别策略**
设计双盲随机对照实验：招募500名即将退役或刚退役的军人，随机分为"身份框架组"（阅读强调军人尊崇身份的《条例》摘要）和"信息中性组"（阅读仅包含政策条款的中性文本）。两组随后完成创业决策博弈任务（涉及不同风险收益结构的项目选择）。通过比较两组的风险偏好差异识别身份框架的因果效应。辅以工具变量法：以服役部队的"荣誉传统强度"（所在师/旅历史荣誉称号数量）作为身份认同强度的工具变量。

**3. 计量模型设定**
实验分析模型：$RiskScore_i = \alpha + \beta_1 IdentityFrame_i + \beta_2 Ability_i + \beta_3 MilYears_i + \gamma X_i + \epsilon_i$，其中$RiskScore_i$为个体$i$在创业博弈中选择的风险水平（0-100量表），$IdentityFrame_i$为身份框架处理变量（1=身份框架组，0=控制组），$Ability_i$为认知能力测试得分，$MilYears_i$为服役年限。交互项$\beta_4 IdentityFrame_i \times LowAbility_i$检验H2（身份框架对低能力者的过度激励）。

**4. 变量构造与数据**
实验数据：①风险偏好得分（Holt-Laury风险量表，10组二选一决策）；②创业项目选择（4个模拟项目，风险-收益递增）；③过度自信测度（自信评分与实际能力测试得分之差）。调查数据：身份认同强度（5点李克特量表）、社会规范感知（"您认为社会对退役军人创业的期待程度"）、创业意愿。档案数据：退役军人创业企业的3年存活率（市场监管总局注销数据）。

**5. 稳健性检验方案**
①需求效应控制：在实验末尾加入"实验目的猜测"问卷，排除参与者迎合实验者预期的行为；②外部效度验证：用CFPS退役军人子样本的真实创业数据，检验实验室发现是否与田野观察一致；③替代框架实验：增加"理性计算框架"组（强调创业的财务风险与失败率），比较三种框架的效应差异；④异质性分析：按军种、军衔、教育水平分组，检验框架效应的边界条件；⑤长期跟踪：实验后12个月回访参与者的实际创业行为，评估实验室效应的持续性。

#### 5. ⚖️ 政治经济学(国家-市场/央地博弈) — 退役军人就业优待的央地博弈：政策执行偏差与地方财政约束

| | |
|---|---|
| **研究问题** | 《条例》规定的就业优待措施（公务员定向招录、国企优先招聘等）在中央-地方委托代理框架下会产生多大的执行偏差？地方财政约束如何系统性地影响退役军人优待政策的落地程度？ |
| **理论框架** | 委托代理理论（Holmström, 1979）、中国式财政联邦主义（Montinola et al., 1995; Qian & Weingast, 1997）、街头官僚理论（Lipsky, 1980）、多任务激励模型（Holmström & Milgrom, 1991） |
| **研究方法** | 文本分析法（比较央地政策文本差异）+ 面板固定效应模型 + 案例比较研究 |
| **数据来源** | 中央与31省退役军人就业创业政策文本数据库（北大法宝）、各省公务员招录公告中国退军人定向岗位数量（各省人事考试网）、地方财政决算数据（财政部）、各省退役军人事务厅信访与投诉数据 |
| **创新点** | 构建"央地政策文本相似度"指标量化政策执行偏差，在多任务激励框架下解释地方政府对退役军人优待政策的"选择性执行"逻辑 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于Holmström & Milgrom（1991）多任务激励理论和Qian & Weingast（1997）中国式财政联邦主义，中央政府将退役军人就业创业纳入地方考核体系，但地方政府面临多任务约束（经济增长、社会稳定、财政平衡等）：①"硬指标优先"——GDP考核压力下，地方政府对退役军人就业优待的执行力度弱于经济指标（H1：财政压力越大的地区，公务员定向招录比例越低）；②"政策文本变通"——省级实施细则通过模糊措辞保留执行弹性（H2：央地政策文本相似度越低，实际优待覆盖率越低）；③"街头官僚裁量"——基层执行者（人社局工作人员）的裁量权导致政策"最后一公里"偏差（H3：县区级执行差异大于省市级）。

**2. 识别策略**
利用31个省级行政区在《条例》出台后发布配套实施细则的时间差异和内容差异，构建"政策文本相似度"指数（基于TF-IDF余弦相似度，计算省级细则与中央《条例》关键条款的语义距离）。以该相似度指数为核心解释变量，以各省退役军人实际就业优待覆盖率为被解释变量，建立面板模型。内生性处理：以各省1985年百万大裁军安置人数作为工具变量——历史安置任务影响地方退役军人制度传统但不直接影响当前就业市场。

**3. 计量模型设定**
基准模型：$Coverage_{pt} = \alpha + \beta_1 SimIndex_{pt} + \beta_2 FiscalGap_{pt} + \beta_3 SOERatio_{pt} + \gamma X_{pt} + \mu_p + \lambda_t + \epsilon_{pt}$，其中$Coverage_{pt}$为省份$p$的退役军人就业优待覆盖率（享受优待人数/符合资格人数），$SimIndex_{pt}$为政策文本相似度指数，$FiscalGap_{pt}$为财政收支缺口率，$SOERatio_{pt}$为国有经济就业占比，$X_{pt}$为控制变量（人均GDP、失业率、退役军人密度），$\mu_p$和$\lambda_t$分别为省份和时间固定效应。

**4. 变量构造与数据**
被解释变量：退役军人就业优待覆盖率=（公务员定向招录退役军人人数+国企优先招聘退役军人人数+享受创业税收减免人数）/ 当年退役总人数。核心解释变量：政策文本相似度指数——利用Python自然语言处理工具，计算各省实施细则与中央《条例》在"招录""培训""税收""融资"四个关键词群的语义相似度（0-1指数）。控制变量：地方财政收支缺口率、国有企业就业占比、城镇登记失业率、人均GDP、退役军人占常住人口比例。工具变量：1985年各省安置裁军人数。

**5. 稳健性检验方案**
①替换相似度度量：使用BERT嵌入向量的余弦相似度替代TF-IDF方法，验证语义距离测度的稳健性；②控制政治因素：加入省委书记/省长任期、年龄（晋升激励代理变量），排除政治锦标赛效应的干扰；③空间溢出检验：用空间杜宾模型检验邻省政策执行的溢出效应（"标杆竞争"或"逐底竞争"）；④微观验证：在3个代表性省份（高/中/低财政压力）开展问卷调查，直接测量退役军人的政策感知与实际享受待遇的差距；⑤反事实分析：模拟如果所有省份达到最高执行标准，退役军人就业率和收入将提升多少。
