#### 1. 📊 新古典(市场均衡/资源配置) — 算力基础设施公共投资与市场均衡：基于智算集群建设的福利分析

| | |
|---|---|
| **研究问题** | 政府主导的超大规模智算集群建设如何影响AI算力市场的均衡价格、企业进入壁垒与社会福利分配？ |
| **理论框架** | Arrow-Debreu一般均衡理论、公共品供给理论（Samuelson, 1954）、自然垄断定价理论（Baumol et al., 1982） |
| **研究方法** | 可计算一般均衡（CGE）模型 + 反事实仿真分析 |
| **数据来源** | 中国信通院算力发展报告、各省市智算中心投资项目数据库、A股AI算力企业财报、国家统计局投入产出表 |
| **创新点** | 将算力视为准公共品，构建包含政府算力供给部门的DSGE模型，量化公共智算投资对私人部门挤出/挤入效应 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于Samuelson公共品理论与Baumol自然垄断理论，智算集群具有高固定成本、低边际成本特征，政府投资可能产生三种效应：①降低中小企业算力获取成本的"挤入效应"（H1）；②挤出私人算力投资的"替代效应"（H2）；③通过价格补贴扭曲市场均衡的"扭曲效应"（H3）。福利分析需权衡消费者剩余增加与生产者剩余损失。

**2. 识别策略**
利用2023-2026年各省市智算中心建设的时间差异与规模差异，构建多期DID模型。处理组为获批建设国家级智算集群的城市，对照组为申请未获批城市。关键识别假定：智算集群选址与城市AI产业发展趋势不相关（可通过工具变量"城市电力容量"验证外生性）。

**3. 计量模型设定**
基准模型：$Y_{it} = \alpha + \beta_1 SmartCenter_{it} + \beta_2 X_{it} + \mu_i + \lambda_t + \epsilon_{it}$，其中$Y_{it}$为城市$i$在$t$期的AI企业数量/算力价格/创新产出，$SmartCenter_{it}$为智算集群建设的强度指标（投资额/GPU算力规模），$X_{it}$为控制变量（GDP、人口、高校数量、互联网普及率），$\mu_i$为城市固定效应，$\lambda_t$为时间固定效应。

**4. 变量构造与数据**
被解释变量：①AI企业新注册数（天眼查数据）；②云计算API调用价格指数（阿里云/腾讯云公开报价）；③AI专利授权量（CNIPA数据库）。核心解释变量：智算集群投资强度（亿元/万人）。控制变量：城市GDP、人口规模、高校数量、互联网带宽、R&D支出占比。工具变量：城市历史电力装机容量（1990年数据）。

**5. 稳健性检验方案**
①替换被解释变量：用AI论文发表量替代专利数量；②更换识别策略：采用断点回归（RDD），以智算集群审批的投资门槛值为断点；③排除竞争性解释：控制同期其他产业政策（如数字经济试点城市）；④安慰剂检验：随机分配处理组500次，生成伪估计系数分布；⑤样本选择：剔除北上广深一线城市，避免极端值影响。

#### 2. 💰 凯恩斯(总需求/政策乘数) — "十五五"碳达峰行动的财政乘数效应：绿色投资的宏观经济刺激机制

| | |
|---|---|
| **研究问题** | 碳达峰行动方案中的公共绿色投资是否具有高于传统基建的财政乘数？其传导机制与条件依赖性如何？ |
| **理论框架** | Keynes财政乘数理论（Kahn, 1931）、绿色凯恩斯主义（Pollitt, 2020）、零利率下限下的乘数放大效应（Christiano et al., 2011） |
| **研究方法** | 结构向量自回归（SVAR）+ 地方绿色投资冲击的叙事法识别 |
| **数据来源** | 各省市政府绿色投资预算数据、CEIC宏观经济数据库、中国碳排放数据库（CEADs）、WIND金融终端 |
| **创新点** | 首次估算中国碳达峰政策的动态财政乘数，区分"绿色投资"与"碳约束"两类冲击的非对称效应 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于Keynes乘数理论与Christiano et al.（2011）的ZLB乘数放大机制，碳达峰投资具有双重属性：①短期需求侧——绿色基建投资直接拉动总需求，产生乘数效应（H1：绿色投资乘数>1.5）；②中期供给侧——碳约束提高传统能源成本，产生"成本推动型收缩"（H2：碳约束冲击短期为负向供给冲击）；③条件依赖：在经济下行期，绿色投资乘数因闲置产能而放大（H3）。

**2. 识别策略**
采用叙事法（Narrative Approach, Romer & Romer, 2010）识别外生绿色投资冲击：从国务院、发改委文件中提取"非经济周期驱动"的碳达峰政策变动（如技术性减排目标调整），排除"逆周期调节"性质的绿色投资。构建SVAR模型，设定递归识别次序：绿色投资冲击→产出→通胀→利率→传统投资。

**3. 计量模型设定**
SVAR模型：$A_0 Y_t = A_1 Y_{t-1} + ... + A_p Y_{t-p} + \epsilon_t$，其中$Y_t = [\Delta GreenInvest_t, \Delta GDP_t, \Delta CPI_t, R_t, \Delta PrivateInvest_t]'$。绿色投资冲击$\epsilon_t^{green}$通过叙事法识别的外生政策变动序列构造。脉冲响应函数追踪10年动态路径，方差分解量化绿色投资冲击对产出波动的贡献度。

**4. 变量构造与数据**
绿色投资：各省"节能环保支出"+"可再生能源补贴"+"绿色债券发行额"（财政部+Wind数据库）。GDP：季度实际GDP（CEIC）。CPI：消费者价格指数。利率：SHIBOR 3个月期。私人投资：固定资产投资完成额-政府投资部分。碳达峰政策冲击：手工收集2015-2026年国务院/发改委碳达峰政策文件，按"预期外投资增量"编码为叙事序列。

**5. 稳健性检验方案**
①替换识别方法：用Blanchard-Quah长期识别限制（绿色投资对GDP长期中性）替代递归识别；②控制财政支出冲击：在SVAR中加入一般公共预算支出变量，分离绿色投资的"纯效应"；③非线性检验：用Threshold-VAR检验经济周期不同阶段乘数的非对称性；④省级面板验证：用省级面板IV模型交叉验证SVAR结果；⑤国际比较：将中国乘数与OECD国家绿色复苏乘数进行对比。

#### 3. 🏛️ 制度经济学(产权/交易成本/激励) — AI数据要素的产权界定与交易成本：中国"高质量数据供给"改革的制度经济学分析

| | |
|---|---|
| **研究问题** | 会议提出"强化高质量数据供给"，数据要素产权界定如何影响AI企业的创新激励？数据交易市场如何降低交易成本？ |
| **理论框架** | Coase产权定理（Coase, 1960）、交易成本经济学（Williamson, 1985）、数据要素市场理论（Acemoglu & Restrepo, 2020） |
| **研究方法** | 双重差分法（DID）+ 案例比较研究（混合方法） |
| **数据来源** | 中国各地数据交易所交易数据（上海/北京/深圳/贵阳等）、AI企业专利与融资数据（天眼查+CVSource）、数据确权政策文本 |
| **创新点** | 将数据要素纳入Williamson交易成本分析框架，构建"数据产权强度→交易成本→AI创新"的因果链条 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于Coase产权定理，清晰的产权界定可降低交易的外部性成本。数据要素具有非竞争性、非排他性特征，产权界定面临独特挑战。研究假说：①数据确权试点城市内AI企业创新产出更高（H1：产权激励效应）；②数据交易所设立降低了AI企业的数据获取成本（H2：交易成本节约效应）；③产权界定对中小AI企业的激励效应更强（H3：规模异质性，因大企业可通过纵向一体化绕过市场交易）。

**2. 识别策略**
利用2021-2026年各地数据交易所设立的时间差异构建多期DID。处理组：所在城市设立数据交易所的AI企业；对照组：未设立数据交易所城市的AI企业。外生性论证：数据交易所选址主要由省级政策驱动，与单个AI企业的创新决策无直接关联。进一步利用数据确权登记制度的"渐进式推进"构造工具变量（城市数字经济试点身份）。

**3. 计量模型设定**
$Innovation_{ijt} = \alpha + \beta_1 DataExchange_{jt} + \beta_2 (DataExchange_{jt} \times Small_i) + \gamma X_{ijt} + \delta Z_{jt} + \mu_i + \lambda_t + \epsilon_{ijt}$，其中$Innovation$为企业$i$在城市$j$、时间$t$的创新产出（专利申请数/软件著作权数），$DataExchange$为数据交易所运营强度指标，$Small$为中小企业虚拟变量，$X$为企业控制变量，$Z$为城市控制变量。

**4. 变量构造与数据**
被解释变量：AI企业年度发明专利申请量、软件著作权登记数、融资轮次进展。核心解释变量：城市数据交易所交易额/GDP占比、数据确权登记数量。控制变量：企业规模（员工数/营收）、年龄、R&D投入、城市GDP、互联网普及率、高校AI学科数量。数据确权政策虚拟变量：根据各地数据条例颁布时间编码。数据来源：天眼查企业数据库、CVSource融资数据库、CNIPA专利数据库、各数据交易所年报。

**5. 稳健性检验方案**
①PSM-DID：用倾向得分匹配构造平衡样本后再做DID；②替换处理变量：用"数据确权登记数量"替代"数据交易所设立"；③排除同期政策干扰：控制"数字经济创新发展试验区""人工智能创新应用先导区"等同期政策；④机制检验：中介效应模型检验"交易成本"（以数据采购成本占营收比重代理）的传导渠道；⑤案例研究：选取上海数据交易所与贵阳大数据交易所进行深度比较分析。

#### 4. 🌏 发展经济学(结构转型/技术追赶) — "人工智能+"战略与后发国家技术追赶：中国AI产业政策的结构转型效应

| | |
|---|---|
| **研究问题** | "人工智能+"行动能否帮助中国跨越中等收入陷阱中的"技术追赶窗口"？AI技术扩散如何影响产业结构转型？ |
| **理论框架** | 新结构经济学（Lin, 2012）、技术追赶理论（Abramovitz, 1986）、偏向性技术进步（Acemoglu, 2002）、雁阵模型（Akamatsu, 1962） |
| **研究方法** | 跨国面板回归 + 中国省级面板GMM估计 |
| **数据来源** | World Bank WDI、UN Comtrade、OECD STAN数据库、中国工业统计年鉴、各省AI产业政策文本编码 |
| **创新点** | 将AI技术嵌入后发国家结构转型模型，区分"AI替代效应"与"AI增强效应"对产业升级的差异化路径 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于Abramovitz社会能力理论与Lin新结构经济学，后发国家的AI技术追赶受制于"社会能力"门槛。假说：①AI技术引进对中等收入国家产业结构升级有显著正向效应，但存在人力资本门槛（H1：门槛效应）；②"AI+"战略通过提升制造业全要素生产率促进结构转型（H2：TFP渠道）；③中国AI产业政策的效应强于其他金砖国家，源于"产业体系完备"的比较优势（H3：互补性效应）。

**2. 识别策略**
跨国层面：利用2010-2025年50个国家的面板数据，以各国AI专利申请量的外生变动（由全球AI技术前沿突破引发的"技术溢出"）作为工具变量。国内层面：利用各省"人工智能+"政策出台时间差异构造多期DID，并以省内高校AI专业设立时间作为工具变量。跨国与国内两层识别相互验证。

**3. 计量模型设定**
跨国面板：$StructChange_{it} = \alpha + \beta_1 AIIntensity_{it} + \beta_2 (AIIntensity_{it} \times HumanCapital_{it}) + \gamma X_{it} + \mu_i + \lambda_t + \epsilon_{it}$。省级面板GMM：$\Delta lnTFP_{it} = \alpha + \beta_1 AIPolicy_{it} + \beta_2 \Delta lnK_{it}/L_{it} + \gamma X_{it} + \mu_i + \lambda_t + \epsilon_{it}$，用系统GMM控制内生性。结构转型指标：制造业增加值占GDP比重、高技术产业占比、产业复杂度指数（Hausmann et al., 2007）。

**4. 变量构造与数据**
结构转型指标：①制造业增加值占比（WDI）；②出口产品复杂度EXPY（UN Comtrade）；③高技术产业营收占比（中国工业统计年鉴）。AI强度：①AI专利占比（WIPO）；②AI论文发表量（Scopus）；③工业机器人密度（IFR）。人力资本门槛：高等教育入学率、STEM专业毕业生占比。控制变量：GDP per capita、贸易开放度、FDI流入、制度质量（WGI）、基础设施。

**5. 稳健性检验方案**
①替换门槛变量：用互联网普及率、电力消费替代人力资本作为社会能力代理变量；②排除反向因果：用滞后5期的全球AI技术冲击作为IV；③子样本检验：按收入水平分组（高收入/中高收入/中低收入），验证门槛效应非线性；④合成控制法：为中国AI战略构造"合成对照组"（其他未实施大规模AI战略的中高收入国家加权组合）；⑤产业层面微观验证：用上市公司数据分析AI技术采用对企业生产率的影响。

#### 5. 👷 劳动经济学(工资/就业/人力资本) — AI规模化商业应用的劳动力市场效应：基于"人工智能+"行动的技能偏向性分析

| | |
|---|---|
| **研究问题** | "人工智能+"行动推动的规模化商业应用如何影响不同技能劳动者的就业与工资？是否存在技能极化效应？ |
| **理论框架** | 技能偏向性技术进步（Acemoglu & Autor, 2011）、任务模型（Autor et al., 2003）、AI替代-增强框架（Acemoglu & Restrepo, 2019） |
| **研究方法** | Bartik工具变量法 + 省级-行业面板回归 |
| **数据来源** | 中国劳动统计年鉴、CFPS微观调查数据、BOSS直聘/智联招聘职位数据、各省AI企业注册数据（天眼查） |
| **创新点** | 利用中国"人工智能+"政策的外生冲击，检验AI对"常规-非常规"任务的差异化替代效应，填补发展中国家AI劳动市场效应研究空白 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于Acemoglu & Restrepo（2019）任务模型，AI技术对不同任务类型产生差异化影响：替代"常规认知型"任务、增强"非常规分析型"任务、暂时难以影响"手工型"任务。假说：①AI商业化应用提高高技能劳动者工资溢价（H1：技能偏向效应）；②AI对中等技能常规岗位产生替代效应，导致就业极化（H2：空心化效应）；③AI创造新任务（如数据标注、AI训练师），部分抵消替代效应（H3：再平衡效应）。

**2. 识别策略**
采用Bartik工具变量法（Shift-Share Design）：利用全国层面各职业AI可替代概率（参照Felten et al., 2021的AIOE指标）与城市基期职业结构的交互项构造城市层面的"AI暴露度"。识别假定：城市基期职业结构不受当期AI政策冲击影响（历史产业结构的延续性）。同时利用"人工智能+"试点城市的政策冲击作为第二重识别。

**3. 计量模型设定**
城市-行业面板：$\Delta Employment_{ijt} = \alpha + \beta_1 AIExposure_{ij} + \beta_2 SkillLevel_j + \beta_3 (AIExposure_{ij} \times SkillLevel_j) + \gamma X_{it} + \mu_i + \lambda_j + \epsilon_{ijt}$。工资方程：$ln Wage_{it} = \alpha + \beta_1 AIExposure_i + \beta_2 Education_i + \beta_3 (AIExposure_i \times Education_i) + \gamma X_i + \epsilon_i$，用2SLS估计，Bartik IV为$Z_i = \sum_j \frac{L_{ij,2018}}{L_{i,2018}} \times AIOE_j$。

**4. 变量构造与数据**
被解释变量：①城市-职业层级就业人数变化率（劳动统计年鉴）；②个体工资收入（CFPS）；③新招聘职位数量与技能要求（BOSS直聘爬虫数据）。核心解释变量：AI暴露度指数（Bartik构造法）。技能分类：高技能（本科及以上+非常规分析型）、中等技能（大专+常规认知型）、低技能（高中及以下+手工型）。控制变量：城市GDP增长率、产业结构、最低工资标准、高校数量、互联网普及率。

**5. 稳健性检验方案**
①替换AI暴露度度量：用中国本土AI企业密度（每万家企业中AI企业数）替代Bartik IV；②分位数回归：检验AI对工资分布不同分位点的异质性效应；③动态效应分析：事件研究法检验"人工智能+"试点前后就业结构的动态变化路径；④排除贸易冲击：控制城市出口暴露度（Autor et al., 2013的中国进口竞争冲击框架）；⑤微观验证：用CFPS个体面板数据追踪同一劳动者在AI冲击前后的职业流动与收入变化。

### 🎓 论文选题

#### 1. 📊 新古典(市场均衡/资源配置) — 学龄人口变动下的教育资源最优配置：基于"十五五"规划前瞻布局的福利分析

| | |
|---|---|
| **研究问题** | 学龄人口下降与区域流动背景下，教育资源的空间再配置能否实现帕累托改进？"前瞻布局"政策对教育公平与效率的均衡效应如何？ |
| **理论框架** | Tiebout模型（Tiebout, 1956）、公共品空间配置理论（Oates, 1969）、教育资源生产函数（Hanushek, 1979） |
| **研究方法** | 空间均衡模型 + 反事实仿真分析 |
| **数据来源** | 第七次人口普查学龄人口数据、教育部全国教育事业发展统计公报、各县区学校布局数据（教育部基础教育质量监测中心）、地方教育财政支出决算数据 |
| **创新点** | 将学龄人口变动纳入Tiebout空间均衡框架，构建"人口流动-学校撤并-教育质量"三部门模型，量化前瞻布局的福利效应 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于Tiebout（1956）"用脚投票"理论，家庭根据公共服务质量选择居住地，教育资源的空间配置影响人口流动均衡。在学龄人口下降背景下，学校撤并可能产生规模经济效应但也增加通勤成本。假说：①前瞻布局通过学校合并提升生均教育质量（H1：规模经济效应）；②学校撤并对农村家庭产生福利损失，加剧城乡教育不平等（H2：空间不平等效应）；③最优学校密度存在阈值，低于阈值后撤并的边际成本超过边际收益（H3：非线性效应）。

**2. 识别策略**
利用2016-2026年各县区学校撤并的时空差异，构建多期DID模型。处理组为实施大规模学校布局调整的县区，对照组为保持学校布局基本不变的相邻县区。关键识别假定：学校撤并决策主要由学龄人口变动驱动，与不可观测的教育质量趋势不相关。利用"二孩政策"实施后的出生率外生变动作为工具变量，缓解内生性问题。

**3. 计量模型设定**
$EduQuality_{ijt} = \alpha + \beta_1 SchoolMerge_{jt} + \beta_2 (SchoolMerge_{jt} \times Rural_{ij}) + \gamma X_{ijt} + \delta Z_{jt} + \mu_j + \lambda_t + \epsilon_{ijt}$，其中$EduQuality$为学生$i$在学校$j$、时间$t$的学业成绩/升学率，$SchoolMerge$为学校撤并强度指标（撤并学校数/原有学校数），$Rural$为农村学生虚拟变量，$X$为学生与家庭控制变量，$Z$为县区教育财政变量。

**4. 变量构造与数据**
被解释变量：①生均教育支出效率（教育支出/在校学生数）；②初中升学率与高考本科达线率；③学生标准化测试成绩（教育质量监测数据）。核心解释变量：县区学校撤并强度（撤并学校比例）。控制变量：县区人均GDP、财政教育支出占比、师生比、学龄人口密度、城镇化率、流动人口比例。工具变量：县区2016年"全面二孩"政策后的出生率变化。

**5. 稳健性检验方案**
①替换被解释变量：用教师流动率、家长满意度调查数据替代学业成绩；②空间断点回归：以学校撤并的地理边界为断点，比较边界两侧学生的教育产出差异；③动态效应检验：事件研究法追踪学校撤并前后5年教育指标的动态演变路径；④排除同期政策干扰：控制"义务教育优质均衡发展"督导评估政策；⑤样本选择偏差修正：Heckman两阶段模型处理学校撤并非随机选择问题。

#### 2. 🏛️ 制度经济学(产权/交易成本/激励) — 教育评价改革的激励相容机制：制度变迁视角下"十五五"教育治理效能研究

| | |
|---|---|
| **研究问题** | "深化教育评价改革"如何改变地方政府、学校与教师的激励结构？"破五唯"政策的制度绩效与交易成本变化如何评估？ |
| **理论框架** | 机制设计理论（Hurwicz, 1973; Maskin, 1999）、委托-代理模型（Holmström, 1979）、制度变迁理论（North, 1990） |
| **研究方法** | 断点回归（RDD）+ 多案例制度分析（混合方法） |
| **数据来源** | 教育部教育评价改革试点县区名单、各省教育督导评估数据、中国教师发展报告（教育部教师工作司）、高校科研评价制度改革文件汇编 |
| **创新点** | 将教育评价改革嵌入Holmström多任务委托-代理模型，分析"破五唯"后多维考核指标对教师行为的激励扭曲效应 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于Holmström（1979）多任务委托-代理理论，当教师承担教学、科研、社会服务等多重任务时，单一量化考核（如论文数量）导致努力偏向可测度任务。"破五唯"改革引入多维评价体系，理论上可改善激励相容但增加了考核的交易成本。假说：①评价改革试点地区的教师教学投入显著增加（H1：激励纠偏效应）；②多维评价增加了行政交易成本，产生"评价疲劳"（H2：交易成本效应）；③改革效果受地方执行能力的调节，制度环境较好的地区绩效更优（H3：制度互补性）。

**2. 识别策略**
利用教育部2021-2025年分批设立的教育评价改革试点县区构造断点回归设计。以试点县区的遴选评分门槛值为断点，比较刚好入选与刚好落选县区在教师行为与教育产出上的差异。识别假定：遴选评分在门槛值附近具有局部随机性（可通过McCrary密度检验验证）。同时收集改革前后政策文本进行制度分析。

**3. 计量模型设定**
RDD模型：$Y_{jt} = \alpha + \beta_1 Pilot_j + \beta_2 Score_j + \beta_3 (Pilot_j \times Score_j) + \gamma X_j + \mu_p + \epsilon_{jt}$，其中$Y_{jt}$为县区$j$在$t$期的教育产出指标，$Pilot$为是否入选试点的虚拟变量（断点处取值），$Score$为遴选评分（running variable），$X$为县区控制变量，$\mu_p$为省份固定效应。带宽选择采用Imbens-Kalyanaraman（2012）最优带宽。

**4. 变量构造与数据**
被解释变量：①教师课堂教学时间占比（教师发展调查数据）；②学生综合素质评价得分变化；③教师流动率（跨校调动比例）。核心解释变量：试点身份虚拟变量。控制变量：县区教育支出/GDP、师生比、城镇化率、教师学历结构、校长任期。制度环境变量：省级政府治理效能指数（中国人民大学政府治理研究中心）。交易成本代理变量：教师年度填写评价表格数量与时间。

**5. 稳健性检验方案**
①更换带宽：分别用0.5倍和2倍最优带宽重新估计，检验结果的带宽敏感性；②多项式阶数检验：分别用线性、二次、三次多项式拟合running variable，确保RDD结果稳健；③安慰剂检验：在真实断点两侧随机设定伪断点1000次，生成伪效应分布；④排除排序操纵：McCrary密度检验验证评分在门槛处无跳跃；⑤多案例验证：选取3-5个试点与非试点县区进行深度访谈与制度比较分析。

#### 3. 🧠 行为经济学(有限理性/锚定/前景理论) — 教育数智化变革中的认知偏差与政策采纳：基于前景理论的"十五五"教育数字化推进研究

| | |
|---|---|
| **研究问题** | 教育数智化变革中，教师与管理者对新技术采纳是否存在系统性认知偏差？"损失规避"与"现状偏见"如何影响教育数字化政策落地效果？ |
| **理论框架** | 前景理论（Kahneman & Tversky, 1979）、现状偏见（Samuelson & Zeckhauser, 1988）、技术采纳行为模型（Venkatesh et al., 2003） |
| **研究方法** | 田野实验（随机对照试验）+ 问卷调查 |
| **数据来源** | 教育部教育信息化试点学校数据、自主设计的教师数字化行为调查（覆盖东中西部6省）、学校信息化设备使用日志数据 |
| **创新点** | 将前景理论的"参照点-损失规避"框架应用于教育技术采纳领域，揭示教师"数智化抗拒"的行为根源与干预策略 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于Kahneman & Tversky（1979）前景理论，个体决策以参照点为锚、对损失的敏感度高于同等收益。教师面对教育数智化变革时，倾向于以传统教学模式为参照点，将新技术视为可能的"损失"（教学控制权削弱、技能贬值风险）而非"收益"。假说：①以"损失框架"呈现数字化政策（如"不使用数字化将被淘汰"）比"收益框架"更能推动教师采纳（H1：框架效应）；②教龄越长的教师现状偏见越强，数字化采纳意愿越低（H2：经验锚定效应）；③同伴示范效应可部分克服现状偏见（H3：社会规范干预）。

**2. 识别策略**
在教育信息化试点学校中开展随机对照试验（RCT）。将300所中小学随机分为三组：①损失框架组（强调不采纳数字化的风险与成本）；②收益框架组（强调数字化提升教学效果的收益）；③对照组（中性信息）。干预持续一学期，追踪教师数字化教学工具的使用频率与深度。辅以分层随机抽样调查，检验教龄、学科、地区的异质性效应。

**3. 计量模型设定**
RCT基准模型：$DigitalAdopt_{ist} = \alpha + \beta_1 LossFrame_s + \beta_2 GainFrame_s + \gamma X_{ist} + \mu_d + \epsilon_{ist}$，其中$DigitalAdopt$为学校$s$教师$i$在$t$期的数字化采纳指标（使用频率/使用深度评分），$LossFrame$和$GainFrame$为处理组虚拟变量，$X$为教师个体特征（教龄、学历、学科、信息化培训经历），$\mu_d$为地区固定效应。异质性分析加入交互项$\beta_3 LossFrame_s \times Tenure_i$。

**4. 变量构造与数据**
被解释变量：①数字化教学工具周均使用次数（学校信息化系统日志数据）；②数字化教学设计质量评分（专家盲评）；③学生对数字化教学的满意度。核心解释变量：信息框架处理组虚拟变量。调节变量：教师教龄、学历、以往信息化培训小时数、学校信息化设备配置水平。控制变量：学校规模、城乡属性、生师比、学校管理层支持力度问卷得分。数据来源：自主RCT实验+教育部教育信息化监测平台。

**5. 稳健性检验方案**
①双盲设计：教师不知道自己处于哪个处理组，评估者不知道分组情况；②长期追踪：干预结束后继续追踪2个学期，检验框架效应的持续性；③替换度量方式：用客观使用日志数据替代主观问卷自评；④排除experimenter demand效应：加入"社会期望量表"控制教师"迎合实验者"的行为倾向；⑤外部有效性检验：将实验结果与非实验地区的教育数字化推进数据进行比较，评估实验结论的外推性。

#### 4. ⚖️ 政治经济学(国家-市场/央地博弈) — 教育强国建设中的央地博弈与财政分担：中央教育规划目标与地方执行偏差的政治经济学分析

| | |
|---|---|
| **研究问题** | 中央教育规划目标在地方执行中产生多大偏差？央地教育事权与支出责任划分如何影响政策执行效果？教育财政的"中央请客、地方买单"现象如何治理？ |
| **理论框架** | 委托-代理理论下的央地关系（Shirk, 1993）、晋升锦标赛（周黎安, 2007）、财政联邦主义（Oates, 1999; Qian & Weingast, 1997） |
| **研究方法** | 文本分析法（政策执行偏差量化）+ 省级面板固定效应模型 |
| **数据来源** | 各省教育事业发展"十四五"/"十五五"规划文本、教育部教育督导报告、各省教育经费统计年鉴、省级官员任期与晋升数据（CCP官员数据库） |
| **创新点** | 构建"中央目标-地方规划-实际执行"三层级政策偏差度量指标，揭示教育领域"目标替代"与"选择性执行"的政治经济学机制 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于周黎安（2007）晋升锦标赛理论与Qian & Weingast（1997）财政联邦主义框架，地方官员在中央考核激励下选择性执行政策。教育投入回报周期长、难以在任期内体现政绩，导致地方官员倾向于将资源投向短期可见的"硬指标"（校舍建设）而忽视"软指标"（教育质量）。假说：①中央教育规划目标在省级规划中被系统性"打折"执行（H1：目标衰减效应）；②GDP导向型官员的教育支出占比低于民生导向型官员（H2：晋升激励扭曲效应）；③中央教育督导能部分纠正地方执行偏差（H3：监督纠偏效应）。

**2. 识别策略**
文本分析法量化政策执行偏差：对比中央《规划》中的量化目标（如师生比、教育支出占比）与省级规划对应指标的差异，构建"目标偏差指数"。计量分析利用省级面板数据，以省级主要领导更替作为教育支出变动的外生冲击（领导人教育理念差异的工具变量）。识别假定：领导人更替时间由中央组织部门决定，与省级教育支出趋势无直接内生关联。

**3. 计量模型设定**
$EduDeviation_{pt} = \alpha + \beta_1 LeaderChange_{pt} + \beta_2 FiscalPressure_{pt} + \beta_3 (LeaderChange_{pt} \times LeaderEdu_{pt}) + \gamma X_{pt} + \mu_p + \lambda_t + \epsilon_{pt}$，其中$EduDeviation$为省份$p$在$t$期的教育政策执行偏差指数（中央目标-实际完成的标准化差距），$LeaderChange$为省级主要领导更替虚拟变量，$LeaderEdu$为新任领导的教育背景特征（是否有教育/社科背景），$FiscalPressure$为财政压力指标。

**4. 变量构造与数据**
被解释变量：①教育政策目标偏差指数（文本分析量化，涵盖10个核心指标的完成度加权得分）；②教育支出占一般公共预算比重与中央要求的差距；③生均教育经费省际差异的基尼系数。核心解释变量：省级主要领导更替、新任领导教育背景。控制变量：省人均GDP、财政收入增长率、转移支付依赖度、城镇化率、学龄人口比例、中央教育督导次数。数据来源：各省规划文本、教育部经费统计年鉴、CCP官员数据库。

**5. 稳健性检验方案**
①替换领导人特征变量：用新任领导的籍贯、年龄、前任岗位类型替代教育背景作为工具变量；②动态效应：事件研究法检验领导更替前后3年教育支出的动态变化路径；③排除经济周期干扰：控制省级GDP增长率与财政收支缺口的交互效应；④跨省比较：按财政自给率将省份分为高/中/低三组，检验财政压力对执行偏差的调节效应；⑤案例研究：选取2-3个省份进行深度政策过程追踪（process tracing），验证计量结果的因果机制。

#### 5. 👷 劳动经济学(工资/就业/人力资本) — "教育科技人才一体化"的人力资本积累效应：基于"十五五"教育规划的工资溢价与就业结构分析

| | |
|---|---|
| **研究问题** | "一体推进教育科技人才发展"能否有效提升人力资本积累速度？教育改革对不同学科、层次毕业生的工资溢价与就业匹配度有何差异化影响？ |
| **理论框架** | 人力资本理论（Schultz, 1961; Becker, 1964）、信号理论（Spence, 1973）、教育-技能匹配理论（Sattinger, 1993）、任务模型（Acemoglu & Autor, 2011） |
| **研究方法** | 双重差分法（DID）+ Oaxaca-Blinder分解 |
| **数据来源** | 中国高校毕业生就业质量年度报告、CHIPs（中国家庭收入项目）微观数据、教育部学科评估数据、BOSS直聘/智联招聘职位大数据 |
| **创新点** | 将"教育-科技-人才"三角关系嵌入人力资本积累模型，区分教育的"人力资本积累效应"与"信号筛选效应"，量化"十五五"重点学科建设对毕业生工资溢价贡献 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于Schultz（1961）人力资本理论与Spence（1973）信号理论，教育改革既可通过提升真实技能（人力资本效应）也可通过改变学历信号强度（信号效应）影响工资。"十五五"规划强调"教育对科技和人才的支撑"，重点学科建设预期提升特定专业的人力资本积累速度。假说：①重点学科建设提升相关专业毕业生的工资溢价（H1：人力资本积累效应）；②"教育-产业"匹配度提高降低毕业生过度教育率（H2：匹配改善效应）；③效应存在学历层次异质性，研究生层次效应大于本科层次（H3：层次梯度效应）。

**2. 识别策略**
利用教育部"双一流"建设与"十五五"高水平应用型本科"双优"建设的时间差异和学科差异构建多期DID。处理组：入选重点学科建设的高校-专业组合；对照组：同一高校未入选专业或相似层次未入选高校的同类专业。识别假定：重点学科遴选在控制高校基期特征后与不可观测的工资趋势无关。Oaxaca-Blinder分解用于区分工资溢价中"可解释部分"（人力资本特征差异）与"不可解释部分"（信号效应/选择效应）。

**3. 计量模型设定**
DID模型：$ln Wage_{igst} = \alpha + \beta_1 KeyDiscipline_{gs} + \beta_2 Post_t + \beta_3 (KeyDiscipline_{gs} \times Post_t) + \gamma X_{igt} + \mu_g + \delta_s + \lambda_t + \epsilon_{igst}$，其中$ln Wage$为毕业生$i$在高校$g$、学科$s$、时间$t$的对数起薪，$KeyDiscipline$为重点学科虚拟变量，$Post$为政策实施后虚拟变量。Oaxaca-Blinder分解：$\Delta ln\bar{W} = (\bar{X}_T - \bar{X}_C)\hat{\beta}_C + \bar{X}_T(\hat{\beta}_T - \hat{\beta}_C)$，区分禀赋差异与回报率差异。

**4. 变量构造与数据**
被解释变量：①毕业生首份工作月薪/年薪（就业质量报告+CHIPs）；②就业岗位与专业匹配度（"完全匹配/部分匹配/不匹配"三类，招聘大数据编码）；③过度教育率（实际学历超过岗位要求学历的比例）。核心解释变量：重点学科建设虚拟变量×政策时间虚拟变量。控制变量：性别、户籍、家庭背景（父母教育年限）、高校层次（985/211/双非）、城市生活成本指数、行业固定效应。数据来源：教育部就业质量报告、CHIPs2018/2023、BOSS直聘API。

**5. 稳健性检验方案**
①PSM-DID：倾向得分匹配构造可比样本后重新估计DID效应；②替换对照组：用"申请未获批"的学科作为对照组替代"同类未申请"学科，减少选择偏差；③区分信号效应与人力资本效应：利用同一专业改革前后"入学分数线变化"作为信号强度代理变量进行中介效应分析；④分位数回归：检验教育改革对工资分布不同分位点（10th/25th/50th/75th/90th）的异质性效应；⑤长期追踪：利用往届毕业生数据追踪改革后3-5年的职业发展轨迹（晋升速度、岗位变动频率）。
