**📄 新闻正文**

新华社长春6月27日电 中共中央书记处书记、国务委员王小洪25日至26日在吉林调研。他强调，要深入学习贯彻习近平总书记重要指示精神，全面落实党中央、国务院决策部署，把握大势、抢抓机遇，强化创新驱动，丰富政策供给，因地制宜发展新质生产力，加快构建现代化产业体系，在高质量发展上迈出坚实步伐，助力推动新时代东北全面振兴。

### 🎓 论文选题

#### 1. 📊 新古典(市场均衡/资源配置) — 东北振兴政策对区域资本配置效率的影响：基于边际产出均等化视角的实证研究

| | |
|---|---|
| **研究问题** | 东北振兴战略实施以来，政策干预是否改善了东北地区资本边际产出、促进了区域间资源配置效率的收敛？ |
| **理论框架** | Hsieh & Klenow (2009) 资源错配理论；Brandt et al. (2013) 中国要素市场扭曲分析；Lucas (1990) 资本流动悖论 |
| **研究方法** | 双重差分法（DID），以东北振兴政策实施为外生冲击，比较东北与其他老工业基地的资本边际产出差异变化 |
| **数据来源** | 中国工业企业数据库、各省份统计年鉴、世界银行中国营商环境调查数据库 |
| **创新点** | 将"新质生产力"概念操作化为资本配置效率的度量指标，检验产业政策能否突破传统要素错配陷阱 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于 Hsieh & Klenow (2009) 的资源错配模型，东北长期受体制性因素导致的要素价格扭曲影响，资本边际产出偏离最优水平。政策冲击若能降低制度性交易成本，将推动资本向高生产率企业重新配置。H1：振兴政策显著提升东北地区资本配置效率；H2：政策效果在国有企业密集行业更为显著；H3：新质生产力导向政策对高技术行业配置效率改善幅度更大。

**2. 识别策略**
以2003年东北振兴战略实施及后续重大政策节点为准自然实验，采用多期DID设计。处理组为东北三省地级市，对照组为其他老工业基地城市（如重庆、武汉等）。核心假定为政策实施前处理组与对照组资本配置效率趋势平行，通过事件研究法验证。同时使用倾向得分匹配（PSM-DID）缓解选择性偏误，确保对照组可比性。

**3. 计量模型设定**
设定多期DID模型：MRPKit = α + β1(Dongbei_i × Post_t) + β2X_it + γ_i + δ_t + ε_it，其中 MRPK 为资本边际产出，Dongbei 为处理组虚拟变量，Post 为政策时间虚拟变量，X 为控制变量（GDP增长率、人口流动、财政支出等），γ_i 和 δ_t 分别为城市和年份固定效应。β1 为核心估计参数，反映政策对资本配置效率的因果效应。

**4. 变量构造与数据**
被解释变量为城市-行业层面资本边际产出（MRPK），基于生产函数估计残差法计算。核心解释变量为政策冲击交互项。控制变量包括：城市GDP、人均收入、固定资产投资额、FDI流入、人力资本（每万人大专以上学历人口）、基础设施（公路密度）。数据来自《中国城市统计年鉴》（2000-2025）、中国工业企业数据库、各省统计年鉴。样本覆盖全国286个地级市。

**5. 稳健性检验方案**
第一，安慰剂检验：随机分配处理组1000次，检验估计系数分布是否集中在零附近。第二，替换对照组：使用合成控制法（Synthetic Control）构建东北的"反事实"样本。第三，更换被解释变量：以全要素生产率（TFP）的离散度作为资本配置效率的替代指标。第四，排除同期政策干扰：控制西部大开发、中部崛起等其他区域政策的交叉影响。第五，使用工具变量（历史铁路密度）处理潜在内生性。

#### 2. 🏛️ 制度经济学(产权/交易成本/激励) — 涉企执法规范化与营商环境优化：知识产权保护如何影响东北企业创新投入

| | |
|---|---|
| **研究问题** | 涉企执法规范化与知识产权保护力度加强对东北地区企业研发投入和创新产出的因果效应是什么？制度性交易成本的降低如何改变企业创新激励？ |
| **理论框架** | North (1990) 制度变迁与产权理论；Williamson (1985) 交易成本经济学；Acemoglu et al. (2005) 制度作为经济增长的根本原因 |
| **研究方法** | 断点回归设计（RDD），利用知识产权保护示范城市政策的地理边界；辅以企业层面面板固定效应模型 |
| **数据来源** | 国家知识产权局专利数据库、中国工业企业数据库、世界银行中国营商环境企业调查、各省市知识产权保护指数 |
| **创新点** | 将"涉企执法规范化"量化为制度性交易成本指标，揭示知识产权保护对东北老工业基地创新激励的异质性传导机制 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于 North (1990) 的制度理论，产权保护不力导致企业创新收益无法内部化，抑制研发投入。东北长期存在执法不规范、产权保护弱等制度短板，增加了企业制度性交易成本。当涉企执法规范化程度提高、知识产权保护力度加大时，企业创新的预期收益上升，从而激励更多研发投入。H1：知识产权保护显著正向影响企业研发强度；H2：制度性交易成本降低对民营企业创新激励效应更大；H3：知识产权保护的效果在技术密集型行业更为突出。

**2. 识别策略**
利用国家知识产权示范城市政策的地理边界实施空间断点回归（Spatial RDD），比较边界两侧相似城市的企业创新行为差异。该策略假定地理边界附近的城市在经济社会特征上具有连续性，政策差异构成准自然实验。同时构建知识产权保护强度的连续指标，使用面板固定效应模型控制城市和时间不可观测因素，以政策冲击的时序变化作为额外识别来源。

**3. 计量模型设定**
RDD 模型：RD_it = α + τ·Treat_i + f(d_i) + γ·(Treat_i × f(d_i)) + X'_itβ + ε_it，其中 RD 为企业研发强度，Treat 为是否位于知识产权保护示范城市，d 为城市到政策边界的地理距离，f(·) 为多项式函数，X 为企业特征控制变量。面板模型：RD_it = α + β·IPR_it + γ·X_it + μ_i + λ_t + ε_it，IPR 为知识产权保护强度指数。

**4. 变量构造与数据**
被解释变量为企业研发投入强度（研发支出/营业收入）和专利产出（发明专利申请数）。核心解释变量为知识产权保护强度指数（参考樊纲市场化指数中知识产权保护分项）。控制变量：企业规模、年龄、所有制、利润率、行业集中度、城市人均GDP。制度性交易成本以企业招待费支出占比衡量。数据来自国家知识产权局专利数据库（2010-2025年）、中国工业企业数据库、各省营商环境评估报告。

**5. 稳健性检验方案**
第一，更换带宽和多项式阶数进行 RDD 敏感性检验，确保断点效应不依赖函数形式选择。第二，McCrary 密度检验排除断点处样本操纵。第三，使用双重差分模型（以示范城市获批时间差异构建多期DID）交叉验证 RDD 结果。第四，引入营商环境评价排名作为知识产权保护的工具变量，处理潜在反向因果。第五，分所有制、分行业进行异质性分析，检验传导机制的稳健性。

#### 3. 🧠 行为经济学(有限理性/锚定/前景理论) — "新质生产力"话语锚定效应：政策信号如何影响东北企业管理者投资决策

| | |
|---|---|
| **研究问题** | "新质生产力"等政策话语是否对企业管理者投资决策产生锚定效应？政策信号的框架效应（framing）如何改变企业对新兴产业的风险偏好？ |
| **理论框架** | Tversky & Kahneman (1974) 锚定启发式；Kahneman & Tversky (1979) 前景理论；Shiller (2005) 叙事经济学与动物精神 |
| **研究方法** | 基于企业调查数据的工具变量法，辅以情景实验（Vignette Experiment）检验锚定效应 |
| **数据来源** | 中国企业营商环境调查、上市公司年报文本数据（管理层讨论与分析）、北大光华-财新智库中国企业经营信心指数 |
| **创新点** | 将政策话语作为行为锚点引入投资决策分析框架，量化"新质生产力"叙事对东北企业战略选择的心理影响机制 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于 Tversky & Kahneman (1974) 的锚定启发式理论，高层政策话语构成显著的信息锚点，影响管理者对新兴产业投资收益的先验判断。前景理论预测，政策信号以"机遇"框架呈现时，管理者更倾向于风险寻求行为。东北企业管理者长期面对政策红利预期，容易形成对政策话语的过度依赖。H1：政策话语提及频率与企业在新兴产业投资强度正相关；H2：政策乐观框架放大企业对高收益的锚定偏差；H3：国有企业受政策锚定效应更强。

**2. 识别策略**
利用不同城市政府工作报告中"新质生产力"相关词汇出现频率的差异作为政策信号强度的外生变异。核心假定为词汇频率差异主要由地方政策传导风格而非经济基本面驱动，通过控制城市GDP、产业结构等变量隔离混淆因素。同时设计情景实验：随机向企业管理者呈现不同框架的政策信息（机遇框架 vs. 风险框架 vs. 中性信息），比较其投资决策差异，直接检验锚定效应。

**3. 计量模型设定**
主模型：Invest_it = α + β1·PolicySignal_ct + β2·Controls_it + γ_i + δ_t + ε_it，其中 Invest 为企业新兴产业投资占比，PolicySignal 为所在城市政策话语强度指数。工具变量模型使用上一级政府工作报告中相关词汇频率作为工具变量。情景实验采用方差分析（ANOVA）和有序Logit模型比较不同处理组的投资意愿差异，估计框架效应的平均处理效应（ATE）。

**4. 变量构造与数据**
被解释变量为企业在战略性新兴产业的投资占比（基于年报披露的分行业投资数据）。核心解释变量为政策话语强度指数，通过文本分析方法统计地方政府工作报告中"新质生产力""创新驱动""战略性新兴产业"等关键词频率。控制变量：企业规模、盈利能力、行业属性、管理者年龄与教育背景。情景实验数据通过在线问卷平台收集，样本覆盖东北地区300家企业中高层管理者。

**5. 稳健性检验方案**
第一，排除政策话语的内生性：使用上级政府报告词频作为工具变量进行2SLS估计。第二，安慰剂检验：用非政策关键词频率替代核心解释变量，预期系数不显著。第三，情景实验采用双盲设计和多种操纵检验（manipulation check），确保锚定效应的内部有效性。第四，控制企业管理者的政治关联程度，排除寻租动机的干扰。第五，跨地区比较（东北 vs. 长三角），验证锚定效应的区域异质性。

#### 4. ⚖️ 政治经济学(国家-市场/央地博弈) — 央地博弈视角下东北振兴政策的执行偏差：激励结构、信息不对称与政策变通

| | |
|---|---|
| **研究问题** | 东北振兴战略从中央决策到地方执行过程中存在怎样的政策偏差？央地之间的激励不相容和信息不对称如何导致政策变通与选择性执行？ |
| **理论框架** | Xu Chenggang (2011) 中国地区分权竞争理论；周黎安 (2007) 晋升锦标赛模型；Shleifer & Vishny (1998) 政府干预与掠夺之手 |
| **研究方法** | 比较案例研究 + 文本分析方法，构建政策文本偏离度指标；辅以省级面板数据的面板固定效应模型 |
| **数据来源** | 国务院东北振兴政策文件汇编、各省市配套实施细则、地方政府工作报告、各省财政预决算报告 |
| **创新点** | 构建"政策文本偏离度"量化指标，系统衡量地方执行与中央意图的偏差程度，揭示东北振兴效果受限的政治经济学根源 |

**🔬 研究思路**

**1. 理论分析与研究假说**
基于 Xu Chenggang (2011) 的地区分权理论，中央与地方存在目标函数差异：中央追求区域均衡与长期增长，地方偏好短期经济增长与官员晋升。信息不对称使得中央难以完全监督地方执行。在晋升锦标赛激励下（周黎安, 2007），地方政府倾向于选择性执行容易出政绩的政策，忽视深层制度改革。H1：地方实施细则与中央政策存在系统性文本偏离；H2：偏离程度与地方官员晋升激励强度正相关；H3：偏离度越高，振兴政策对高质量发展的效果越弱。

**2. 识别策略**
采用文本分析方法量化中央政策文件与地方配套文件之间的语义偏离度。利用不同省份、不同任期内官员的晋升压力差异（以官员年龄、任期、GDP排名等衡量）构建解释变量。核心假定为官员个人特征（年龄、任期等）影响其晋升激励但不直接影响政策执行的初始条件，满足排除性约束。通过比较同一省份不同任期、不同省份同一时期的政策偏离差异，控制省份固有特征的干扰。

**3. 计量模型设定**
偏离度模型：Deviation_pt = α + β1·PromotionIncentive_pt + β2·FiscalCapacity_pt + β3·CentralSupervision_pt + γ_p + δ_t + ε_pt，其中 Deviation 为政策文本偏离度（基于 Word2Vec 语义距离或人工编码评分），PromotionIncentive 为地方主官晋升激励强度（年龄、任期等综合指标），FiscalCapacity 为地方财政能力，CentralSupervision 为中央督查频次。γ_p 和 δ_t 为省份和年份固定效应。

**4. 变量构造与数据**
核心被解释变量为政策文本偏离度：使用自然语言处理（BERT/Word2Vec）计算中央文件与地方实施细则的语义相似度，取1-相似度作为偏离度。核心解释变量为晋升激励指标，参考周黎安（2007）构建综合指标（官员年龄距退休年限、任期、本省GDP在全国排名等）。控制变量：地方财政收入、转移支付依赖度、市场化程度、教育水平。数据来自北大法宝政策数据库、各省政务服务网、CSMAR官员数据库。

**5. 稳健性检验方案**
第一，替换偏离度测量方法：从机器语义分析切换为专家人工编码评分（邀请政策研究专家对文本一致性打分），检验结论一致性。第二，使用官员外生调任（如中央空降干部）作为晋升激励变化的工具变量。第三，排除干扰：控制中央巡视组进驻、环保督察等监督事件的影响。第四，缩小样本至老工业基地省份进行子样本回归，确保结论不受非东北省份异质性干扰。第五，更换因变量为政策实施后的经济绩效指标，检验偏离度的实际后果。

#### 5. 🌏 发展经济学(结构转型/技术追赶) — 东北老工业基地的结构转型困境与技术追赶路径：新质生产力能否跨越"中等技术陷阱"

**🔬 研究思路**

**1. 理论分析与研究假说**
基于 Rodrik (2016) 的结构转型理论，东北面临"过早去工业化"与"技术锁定"双重困境。传统产业（汽车、石化）的路径依赖使得资源难以向高技术产业转移，类似于 Hausmann & Rodrik (2003) 所描述的"成本发现"失败。新质生产力战略通过政策引导资源向新能源、新材料等领域配置，可能突破这一困境。H1：东北产业结构集中度与全要素生产率呈倒U型关系；H2：新兴产业投资对传统产业的创造性破坏效应存在门槛效应；H3：技术追赶效果受本地人力资本存量的调节。

**2. 识别策略**
利用东北各地级市在新兴产业投资强度上的外生差异（以国家级高新区、经开区设立时间作为工具变量）识别结构转型对经济增长的因果效应。核心假定为高新区设立时间主要由国家规划决定而非地方经济状况驱动，满足外生性要求。同时采用空间计量模型（空间杜宾模型SDM），利用城市间地理邻近性检验新兴产业技术扩散的空间溢出效应，控制空间自相关和空间异质性。

**3. 计量模型设定**
结构转型模型：TFP_it = α + β1·StructChange_it + β2·EmergingInvest_it + β3·(StructChange_it × EmergingInvest_it) + γ·X_it + μ_i + λ_t + ε_it。空间模型：TFP_it = ρ·W·TFP_it + β·X_it + θ·W·X_it + μ_i + λ_t + ε_it，其中 W 为空间权重矩阵，StructChange 为产业结构变化指数（基于Krugman专业化指数），EmergingInvest 为新兴产业投资强度。门槛模型采用 Hansen (1999) 面板门槛估计。

**4. 变量构造与数据**
被解释变量为城市层面全要素生产率（基于OP法或LP法估计）。核心解释变量包括：产业结构变化指数（二三产业增加值比重的变化率）、新兴产业投资占比（战略性新兴产业固定资产投资/总投资）。控制变量：人力资本（高等教育人口占比）、基础设施、FDI、研发投入强度、城镇化率。"中等技术陷阱"指标定义为：技术复杂度达到全国中位数后增速显著放缓的二元指标。数据来自《中国城市统计年鉴》《中国工业统计年鉴》（2005-2025）及各省投入产出表。

**5. 稳健性检验方案**
第一，替换全要素生产率估计方法（从OP法到LP法再到ACF法），确保结论不受生产函数估计方法影响。第二，工具变量回归：使用国家级高新区设立的外生时间变异作为新兴产业投资的工具变量。第三，空间模型敏感性检验：更换空间权重矩阵（地理距离、经济距离、邻接矩阵），验证溢出效应稳健性。第四，排除同期政策干扰：控制振兴东北专项转移支付、资源型城市转型试点等政策。第五，将分析单元下沉至县级或企业级，检验微观层面一致性。
